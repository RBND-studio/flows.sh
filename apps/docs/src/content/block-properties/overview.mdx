## Customizable properties

Customizable properties are the properties that you can define and use in your block templates.

<Cards>
  <Card title="String" icon={<String16 />} href="/block-properties/string">

  </Card>
  <Card title="Number" icon={<Number16 />} href="/block-properties/string">
    Basic block property for entering numbers.
  </Card>
  <Card title="Boolean" icon={<Boolean16 />} href="/block-properties/boolean">
    Basic block property for toggling between two states.
  </Card>
  <Card title="Select" icon={<Select16 />} href="/block-properties/select">
    Used for selecting from a list of options.
  </Card>
  <Card title="Array" icon={<Array16 />} href="/block-properties/array">

  </Card>
  <Card title="State memory" icon={<Memory16 />} href="/block-properties/state-memory">
    Store a boolean value in the block's state to track user progress.
  </Card>
  <Card title="Block trigger" icon={<BlockTrigger16 />} href="/block-properties/block-trigger">

  </Card>
  <Card title="Block state" icon={<Block16 />} href="/block-properties/block-state">
    Read a state from another block in the workflow.
  </Card>
