Flows makes it easy to add product adoption experiences into your React app built with [TanStack Start](https://tanstack.com). Integrating Flows into your app enables you to create interactive workflows, user onboarding, guided tours, in-app messaging, and more.

This guide walks you through integrating Flows into your TanStack app using the [React SDK](https://www.npmjs.com/package/@flows/react) and [components package](https://www.npmjs.com/package/@flows/react-components) to get you started.

<Callout>
  Using an AI coding assistant? Install the [Flows install skill](/agent-skills) to let your agent
  set up Flows automatically.
</Callout>

<Step>

## Install the React SDK and components package

You can install the Flows React SDK [from npm](https://www.npmjs.com/package/@flows/react):

```package-install
npm i @flows/react @flows/react-components
```

</Step>

<Step>

## Integrate FlowsProvider in your application

Create a Flows component that wraps your application with the `FlowsProvider`. Then add
the `Flows` component to your root layout.

<Callout>
  Check out the [TanStack template
  project](https://github.com/RBND-studio/flows-sdk/tree/main/examples/react/tanstack) for a
  complete example.
</Callout>

```tsx tab="src/components/flows.tsx"
import { Link } from "@tanstack/react-router";

import { FlowsProvider, type LinkComponentType } from "@flows/react";
import * as components from "@flows/react-components";
import * as tourComponents from "@flows/react-components/tour";
import * as surveyComponents from "@flows/react-components/survey";

import "@flows/react-components/index.css";

export const Flows = ({ children }: { children: React.ReactNode }) => {
  return (
    <FlowsProvider
      organizationId="YOUR_ORGANIZATION_ID" // Find this in Settings > General
      userId="YOUR_USER_ID" // Replace this with user id from your app
      environment="production" // Default environment
      components={{ ...components }}
      tourComponents={{ ...tourComponents }}
      surveyComponents={{ ...surveyComponents }}
      LinkComponent={LinkComponent}
    >
      {children}
    </FlowsProvider>
  );
};

const LinkComponent: LinkComponentType = ({ href, children, className, onClick }) => (
  <Link to={href} className={className} onClick={onClick}>
    {children}
  </Link>
);
```

```tsx tab="src/routes/__root.tsx"
import { Flows } from "~/components/flows";

// Wrap your application with the Flows component from flows.tsx
function RootDocument({ children }) {
  return (
    <html>
      <body>
        <Flows>{children}</Flows>
      </body>
    </html>
  );
}
```

For a full list of supported properties, see the [FlowsProvider documentation](/sdk/react/reference#flowsprovider). For an example of how to create a custom component, see the [TanStack template project](https://github.com/RBND-studio/flows-sdk/tree/main/examples/react/tanstack).

</Step>
